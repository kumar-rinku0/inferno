#### Running Locally

##### First, clone repo.
```bash

git clone https://github.com/kumar-rinku0/rinku-portfolio.git

cd rinku-portfolio

```

##### Now, you can run the development server:

```bash

npm install

npm run dev

```

##### Open [localhost:3000](http://localhost:3000) in your browser to see the result.
